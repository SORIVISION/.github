### 핵심 기능
 **1. 웨어러블 디바이스**
 - *앞에 위험해요!* : 충돌 위험 알림
 - *도움이 필요해* : 긴급 상황 요청
 - *"앞에 멋진 성이 있어요.."* : 자동 상황 설명
 - *"이거 유통기한 언제까지야?"* : 요청 기반 설명

 **2. 보호자 애플리케이션**
  - *어디서나 함께해요* : 실시간 위치 모니터링
  - *언제나 안전해야 하니까* : 긴급 상황 알림
  - *일상을 나눠요* : 일일기록(소리발자국)
